Request a report

Query Parameters

  • Report: required (string)

    The short name of the COUNTER report (e.g., JR1, BR1, BR2, AR1, etc.). This will determine the nature of data to be returned. For example, a JR report will return journal-level usage and an AR report will retrieve article-level usage. View the SUSHI report registry for more information: http://www.niso.org/workrooms/sushi/registry/#reports.

  • Release: required (string - default: 4)

    Example:

    4
  • RequestorID: (string)

    Example:

    1
  • CustomerID: (string)

    Example:

    2
  • BeginDate: (string)

    Example:

    Sat Feb 14 2015 19:00:00 GMT-0500 (EST)
  • EndDate: (string)

    Example:

    Sat Feb 28 2015 19:00:00 GMT-0500 (EST)
  • ItemIdentifier: (string)

    Example:

    journal:issn:1111-1111|journal:issn:2222-2222
  • ItemContributor: (string)

    Example:

    author:orcid:0000-0002-8721-8082
  • ResourceType: (string)

    Example:

    Book
  • Publisher: (string)

    Example:

    University of Example Press
  • Platform: (string)

    Example:

    Platform1
  • MetricTypes: (string)

    Example:

    ft_total|ft_html
  • PubYr: (string)

    Example:

    2001
  • PubYrFrom: (string)

    Example:

    1997
  • PubYrTo: (string)

    Example:

    2015
  • IsArchive: (boolean)
  • Granularity: (string)

    Example:

    Yearly
  • ExcludeZeroUsage: (boolean - default: true)

    Example:

    true
  • Format: (string - default: JSON)

    Example:

    XML
  • Callback: (string)
  • Limit: (integer)

    Example:

    1000
  • Offset: (integer - default: 1)

    Example:

    20
  • OrderBy: (string - default: ItemName:asc)

    Example:

    ft_total:desc
